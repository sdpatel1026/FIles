Cause List

How to

b. Select the Cause list Date from the calendar control.
c. Enter the Captcha (the alphanumeric digits shown on the screen) in the text box provided.

Caveat Search

How to


a. Select High Court and Bench of the Cause list to be displayed.
b. Select the entry from the Court Name Select box, which shows the court number, the judge name and the designation of the judge for displaying the Cause list.
c. Select the Cause list Date from the calendar control.
d. Enter the Captcha (the alphanumeric digits shown on the screen) in the text box provided.
e. Click on the Civil or Criminal button, to accordingly display the Civil or Criminal Cause list of the selected Court and Date.

How to

  • i. Click here to view help video.
  • ii. The Cases can be searched by either clicking on the Advocate Name, Bar Registration Number or Advocate’s Cause List (to view the date wise cause list of Advocate) option buttons.
  • iii. On clicking the Advocate Name option button, enter the Name of the Advocate in the box, in part (minimum 3 characters) or full - e.g. for searching the name of the advocate as “Suresh Mahohar Singh” either Suresh, Manohar, Singh, Sur, Sin etc., can be given.
  • iv. On clicking the Bar Registration Number option button, enter the Bar Registration Number of the advocate, in the box, for e.g. MAH/XXXX/1982 (where MAH is the state code, XXXX is the bar registration number and 1982 is the year).
  • v. Click on either the Pending or Disposed option button, according to the status of the Case (only in case the Advocate Name or Bar Registration Number options have been chosen - refer to step iii and iv above). If you don’t know the exact status of the Case, then click on the Both option button.
  • vi. On clicking the Advocate’s Cause List option button, enter the Bar Registration Number in the box, for e.g. MAH/XXXX/1982 (where MAH is the State Code, XXXX is the Bar Registration Number and 1982 is the Year) and select the Cause List Date from the calendar control.
  • vii. Enter the Captcha (the 6 alphanumeric characters shown on the screen) in the text box provided.
  • viii. In case of Advocate and Bar Registration Number option, Click on the Go button, to get Cases satisfying the given Advocate’s Name or Bar Registration Number condition. Click on the adjacent View button from the list, to see the Case History of the respective Case.
  • ix. In case of Advocate’s Cause List option, Click on Go button, to get the Advocate’s Cause List on the selected date.

How to

  • b. Select High Court and Bench of the Caveat Search to be displayed.
  • c. AnyWhere - In this option, the system will display the Caveat when you enter a part of the Caveator or Caveatee name as the search criteria.
    For example: If the user makes a search using the keyword Ram as search criteria for the Caveator or Cacveatee Name , then the system should display all the matching Caveats like Ramakant or Ganesh Ram Joshi or JaiRam Pant.
  • d. Starting With - In this option, the system will display the Caveat when the search string matches the Caveatee or Caveator Name starting with the search criteria.
  • e. Soundex - In this option, the system will display the Caveat when you enter a part of the Caveatee or Caveator Name, which sounds like or phonetically matches with the search criteria.
  • f. Subordinate Court - In this option, you can search the Caveat using Subordinate Court Name as search criteria. If any of the active Caveats match the search criteria then the relevant details are displayed on the screen. The relevant Caveat is selected, so that it can be tagged with the Case. This tab is displayed only if the user belongs to an Appellate Establishment.
  • g. Caveat No. - In this option, the system will display the Caveat when you enter a part of the Caveat No. and Year, If any of the active Caveats match the search criteria then the relevant details are displayed on the screen.
  • h. Enter the Captcha (the 6 digit alphanumeric chararacters shown on the screen) in the text box provided.
  • i. Click on the Go button, to get list of all the Cases satisfying the given Caveat Search condition.
